## Run Locally

Clone the project

~~~bash
git clone https://github.com/ka9mal6t/voice-chat.git
~~~

Go to the project directory

~~~bash
cd voice-chat
~~~

Install dependencies

~~~bash
pip install -r requirements.txt
~~~

Start the server

~~~bash
python server.py
python client.py
~~~
